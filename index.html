📖 스티커 사용 가이드

🍎 아이폰 (사진 앱 / 카카오톡)
  1. 스티커 카드의 📲 저장 또는 📲 갤러리에 전체 저장 탭
  2. 공유 시트가 뜨면 '이미지 저장'(또는 'Save N Images') 선택 → 사진 앱에 PNG 그대로 저장 (투명 배경 보존)
  3. 사진 앱에서 이미지 열고 공유 → 복사
  4. 카카오톡 대화창에 붙여넣기 → 투명 스티커로 전송!
  5. 또는 메시지 앱 → iMessage 스티커로 등록 가능
🤖 안드로이드 (삼성 키보드)
  1. 스티커 카드의 📲 저장 또는 📲 갤러리에 전체 저장 탭
  2. 공유 시트가 뜨면 '갤러리에 저장'(또는 '내 파일에 저장') 선택
  3. 삼성 키보드 → 클립보드 열기 → ✏️ 편집 → 갤러리에서 PNG 선택 → 📌 핀(고정)
  4. 카카오톡 등 채팅창에서 클립보드를 열고 이미지 탭 → 스티커처럼 전송!
💡 더 좋은 결과를 얻으려면?
  1. 이 도구는 흰색 배경 + 글씨/테두리가 있는 이모티콘 시트에 최적화되어 있습니다.
  2. 격자가 정확히 맞아야 깔끔하게 분할됩니다 — 가로/세로 칸 수를 원본과 맞춰주세요.
  3. 외곽 여백/셀 간격이 미세하게 달라도 스마트 분할이 자동 보정해줍니다.
  4. 모든 처리는 브라우저 안에서만 이루어지며 이미지가 외부로 전송되지 않습니다.
