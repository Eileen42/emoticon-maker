📖 스티커 사용 가이드

🤖 안드로이드 (삼성 키보드)
  1. PNG 다운로드 — 분할된 스티커를 기기에 저장
  2. 삼성 키보드 → 클립보드 열기
  3. 클립보드 우측 상단 ✏️ 편집 → 갤러리에서 PNG 선택 → 📌 핀(고정)
  4. 카카오톡 등 채팅창에서 클립보드를 열고 이미지 탭 → 스티커처럼 전송!
🍎 아이폰 (iMessage / 카카오톡)
  1. PNG 다운로드 — 분할된 스티커를 사진 앱에 저장
  2. 갤러리에서 사진을 열고 공유 → 사진 복사
  3. 카카오톡 대화창에 붙여넣기 → 스티커처럼 전송
  4. 또는 메시지 앱에서 iMessage 스티커로 등록하여 사용!
💡 더 좋은 결과를 얻으려면?
  1. 스티커 모드는 흰색 배경 + 글씨/테두리가 있는 이모티콘 시트에 최적입니다.
  2. AI 누끼 모드는 복잡한 배경에서 인물/캐릭터만 추출할 때 유용합니다.
  3. 격자가 정확히 맞아야 깔끔하게 분할됩니다 — 가로/세로 칸 수를 원본과 맞춰주세요.
  4. 모든 처리는 브라우저 안에서만 이루어지며 이미지가 외부로 전송되지 않습니다.
